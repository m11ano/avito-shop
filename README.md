## Тестовое задание: внутренний магазин мерча

Запуск в контейнере:

```
docker compose up
```

Локальный запуск (потребуется поменять uri подключения к бд):

```
make run
```

Запуск всех тестов:

```
make test
```

Просмотр тестового покрытия:

```
make test-cov
```

Запуск только интеграционных тестов:

```
make test-integration
```
